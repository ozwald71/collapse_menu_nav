Collapsible Menu Navigation

Hello and welcome to my collapsible Menu Navigation. I created this navigation using HTML5, CSS, JavaScript, BootStrap, and jQuery. I used HTML for the basic layout, CSS for styling, JavaScript and jQuery for functions, and BootStrap because I love how easy BootStrap is to make a layout. BootStrap also provides responsive viewing. Try changing the window size. I started coding as a hobby and hopefully will turn it into a career someday.

I'm leaving this as an open source for anyone who feels like using the design. Feel free to go to my home on the web, the link is found under the "projects" tab. Thanks.
